# Student Performance Analysis

## Description
Analyzing student exam performance using Python, Pandas and Matplotlib

## Libraries Used
- NumPy
- Pandas
- Matplotlib

## Analysis Done
- Score distributions
- Gender based analysis
- Test preparation impact
- Parental education impact

## Key Insights
- Males score higher in Math
- Females score higher in Reading & Writing
- Test prep improves scores by 5 points!
- Higher parental education = better scores!
